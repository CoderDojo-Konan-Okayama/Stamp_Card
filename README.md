# Stamp Card
CoderDojo Konan で使用しているスタンプカードを以下の理由で電子化してしまおう、と作りました。

- 紙、インク代がもったいない。
- こちらにも、スタンプのたまっている状況を把握することができ、景品？を無駄に買い置きする必要がなくなる。
- 一番はプログラミングの練習と、興味本位。


### 使い方

#### スタンプを押す
GET `○○○○.○○/stamp_push.php/?id=oo&pass=oo`
